# Med Now (Anthah-Prerana-7.0)

A modern, premium healthcare web application designed with Vanilla HTML, CSS, and JS, aimed at providing an extensive and seamless user experience for managing health needs. 

The application utilizes dynamic interfaces, glassmorphism, responsive design, and smooth CSS micro-animations to deliver a next-gen web experience.

## Features & Capabilities

### 1. Dashboard
- Provides an animated overview of health statistics (Heart Rate, Calories, Sleep).
- Dynamic greeting based on the user's current local time.
- Quick action links to other application utilities.

### 2. Medicine Finder
- **Don't waste your time searching in different sites, we got it covered.**
- Allows searching for specific medications or uploading a prescription image.
- Dynamically generates comparative price ranges across multiple delivery platforms (Blinkit, Instamart, Apollo 24/7, Pharmeasy) based on mock base pricing.
- Shows dynamic overall product ratings (out of 5.0) and total review counts for the searched product.
- **Editable Order Action:** Allows users to review and confirm the product name and price directly in an inline popup module.

### 3. Consultancy Tracker
- **Don't waste your time tracking appointments across different hospitals. We got it covered.**
- Visualizes past and upcoming appointments. 
- Links to external hospital portals for immediate access.

### 4. AI Report Analyzer
- **Don't waste your time trying to decode complex medical jargon. We got it covered.**
- Drag & Drop interface for medical PDFs or images.
- Simulates an AI analysis process with dynamic spinner tools and timeline text updates.
- Renders an easy-to-read "Health Score" dashboard, breaking down biomarkers visually with progress bars and text insights.

### 5. AI Medicine Recommender
- **Don't waste your time figuring out which medicine is best. We got it covered.**
- Contextual question logic: asks about temperature for Fever, pain scale for Headache, etc.
- Displays comprehensive side-by-side card comparisons of up to 10 top medical options, evaluated by:
  - Effectiveness
  - Speed of Relief
  - Safety Profile 
  - Overall Rating 

### 6. App Infrastructure
- **Light/Dark Mode Toggle** integrated cleanly within an animated side-settings panel.
- Notification Drawer to aggregate actionable reminders (like upcoming consultations).
- Fluid SVG "We got it for you" screen transitions when navigating through active app tools.

## Setup
Just open `index.html` in a web browser. No complex build tools or bundlers are required.
